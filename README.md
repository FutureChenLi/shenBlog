# 小沈的个人网站
展示自己的个人博客网站

## 契机和目标
* 1.经历了长时间的前端学习，想把自己的学习成果展现出来，所以想做一个展示自己水平的个人博客网站。	
一开始只要求能把成果做出来就行，后期可能着重点在画面的一些效果演示。	
网站内容主要是分享一些个人经历，和技术文案。	
预想的是前后台分离。后台主要提供接口和处理数据。前端主要是表示。
