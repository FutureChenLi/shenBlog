# 前端页面使用说明

- 使用前提
    + 安装Nodejs
    + 安装mysql

- 安装mysql，创建数据库，执行blog.sql文件创建表。
- 修改dynamic-koa/blog/models/db.js文件中的mysql配置。
- 进入dynamic-koa/blog 执行下面的操作
```shell
npm install
npm start
```
- 打开游览器，输入`localhost:3000`,就可以查看网页
